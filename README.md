# William Ariel Tjandra - Portfolio Website

A modern, responsive portfolio website built with React and Vite, featuring a beautiful gradient background.

## Features

- **Modern Design**: Clean, professional layout with gradient background
- **Responsive**: Works seamlessly on all devices
- **Smooth Scrolling**: Smooth navigation between sections
- **Interactive**: Hover effects and transitions throughout

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Building for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18

- CSS3 with modern animations
- Responsive design
